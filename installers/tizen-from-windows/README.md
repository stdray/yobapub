# YobaPub Tizen Installer (Windows)

Установка виджета YobaPub на Samsung Tizen TV.

## Подготовка телевизора

1. Откройте **Apps** на телевизоре
2. Нажмите на пульте последовательно **1 2 3 4 5**
3. Включите **Developer Mode**
4. Введите IP-адрес компьютера, с которого будете устанавливать
5. Перезагрузите телевизор

## Установка

1. Скачайте `.wgt` файл из [Releases](https://github.com/stdray/yobapub/releases) и положите в эту папку
2. Запустите `install.ps1`:

```powershell
powershell -ExecutionPolicy Bypass -File install.ps1
```

Или правый клик по `install.ps1` -> "Выполнить с помощью PowerShell".

## Что делает скрипт

1. Сканирует локальную сеть и находит Samsung TV
2. Предлагает выбрать устройство
3. Ищет существующий Tizen signing profile на компьютере
4. Если профиля нет — открывает **Certificate Manager** для создания нового (см. ниже)
5. Переподписывает `.wgt` вашим сертификатом
6. Устанавливает и запускает на выбранном ТВ

## Создание сертификата (первый запуск)

При первом запуске на новом компьютере откроется Certificate Manager.

1. Нажмите **[+]** для создания нового профиля
2. Выберите **Samsung** (не Tizen!)
3. Придумайте имя профиля (любое)
4. **Author Certificate** -> **Create a new author certificate**
   - Заполните имя и пароль
   - Войдите в **Samsung Account** (создайте на [account.samsung.com](https://account.samsung.com) если нет)
5. **Distributor Certificate** -> **Create a new distributor certificate**
   - Privilege level: **Partner**
   - В списке DUID выберите ваш телевизор (он должен быть подключен)
6. Нажмите **Finish**
7. Закройте Certificate Manager

Скрипт автоматически подхватит созданный профиль и продолжит установку.

## Обновление

Для обновления скачайте новый `.wgt` файл из Releases, замените старый и запустите `install.ps1` снова. Пересоздавать сертификат не нужно — он сохранен в папке `sdk-data\`.

## Устранение проблем

| Проблема | Решение |
|----------|---------|
| ТВ не найден при сканировании | Проверьте что ТВ в режиме разработчика и указан IP вашего компьютера |
| `PKGMGR_ERROR` при установке | DUID в сертификате не совпадает с ТВ. Удалите папку `sdk-data\` и запустите скрипт заново |
| `SIGNATURE_INVALID` (error 118) | То же — пересоздайте сертификат с правильным DUID |
| Samsung Account login не работает | Используйте VPN, если Samsung Account заблокирован в вашем регионе |
| Certificate Manager не видит ТВ | Убедитесь что ТВ подключен — скрипт делает это автоматически перед запуском CM |

## Структура папки

```
tizen-from-windows/
  install.ps1         -- скрипт установки
  README.md           -- эта инструкция
  *.wgt               -- виджет (скачать из Releases)
  sdk/                -- Tizen SDK (JDK + CLI + Certificate Manager)
  sdk-data/           -- данные и сертификаты (создается автоматически)
```
