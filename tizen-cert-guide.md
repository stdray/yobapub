# Создание сертификата Tizen для установки на телевизор

## 1. Включить режим разработчика на телевизоре

1. Откройте **Apps** на телевизоре.
2. Нажмите последовательно кнопки: **1 2 3 4 5** на пульте.
3. Появится окно **Developer Mode** -- переключите в **ON**.
4. Введите IP-адрес компьютера, с которого будете деплоить.
5. Перезагрузите телевизор.

## 2. Подключить телевизор в Device Manager

1. В Tizen Studio откройте **Tools > Device Manager** (или иконка на панели).
2. Нажмите **Remote Device Manager** (иконка монитора с шестерёнкой).
3. Нажмите **+** (Scan) или добавьте вручную: введите IP телевизора и имя.
4. Переключите **Connection** в **ON** -- статус должен стать **Connected**.

## 3. Создать сертификат

### 3.1. Открыть Certificate Manager

**Tools > Certificate Manager** в Tizen Studio.

### 3.2. Создать профиль

1. Нажмите **+** (создать новый профиль).
2. Выберите **Samsung** (не Tizen).
3. Введите имя профиля (любое, например `my-tv-cert`).

### 3.3. Author Certificate

1. Выберите **Create a new author certificate**.
2. Заполните:
   - **Name** -- ваше имя (любое).
   - **Password** -- придумайте пароль (запомните его!).
3. Нажмите **Next**.

### 3.4. Samsung Account

1. Выберите **Sign in with Samsung Account**.
2. Войдите в свой Samsung-аккаунт (тот же, что на телевизоре).
   - Если аккаунта нет -- создайте на [account.samsung.com](https://account.samsung.com).

### 3.5. Distributor Certificate

1. Выберите **Create a new distributor certificate**.
2. **Privilege level**: Partner.
3. В списке **DUID** должен быть DUID вашего подключённого телевизора.
   - Если DUID не появился автоматически, убедитесь что телевизор подключён (шаг 2).
4. Нажмите **Finish**.

## 4. Сборка и установка

1. Убедитесь, что созданный профиль выбран как активный в Certificate Manager.
2. **Project > Clean**, затем **Run As > Tizen Web Application**.
3. В выпадающем списке устройств выберите ваш телевизор.

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| DUID не появляется | Проверьте подключение телевизора в Device Manager |
| Ошибка подписи / зависание сборки | Пересоздайте профиль сертификата |
| `certificate error` при установке | DUID в сертификате не совпадает с телевизором -- пересоздайте distributor certificate с правильным DUID |
| Samsung Account login не работает | Используйте VPN, если Samsung Account заблокирован в вашем регионе |
